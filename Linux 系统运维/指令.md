| 工具                     | 在运维的经典用途                     |
| ---------------------- | ---------------------------- |
| **sort**               | 排序（CPU、内存、磁盘、日志重复）           |
| **xargs**              | 批量执行命令（rm、kill、ping、grep、cp） |
| **sort + uniq + sort** | 统计次数（IP、错误日志）                |
| **find + xargs**       | 批量操作文件                       |
| **awk + sort + xargs** | 实际生产排障组合拳                    |

grep "ERROR" app.log | sort | uniq -c | sort -nr | head
