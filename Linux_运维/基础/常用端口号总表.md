# 一、基础协议类端口

| 服务    | 端口  | 协议      | 说明           |
| ----- | --- | ------- | ------------ |
| HTTP  | 80  | TCP     | Web 服务       |
| HTTPS | 443 | TCP     | 加密 Web       |
| SSH   | 22  | TCP     | Linux 远程登录   |
| DNS   | 53  | UDP/TCP | 域名解析         |
| FTP   | 21  | TCP     | 控制通道         |
| SFTP  | 22  | TCP     | 基于 SSH 的文件传输 |
| NTP   | 123 | UDP     | 时间同步         |
| SMTP  | 25  | TCP     | 邮件发送         |
| POP3  | 110 | TCP     | 邮件接收         |
| IMAP  | 143 | TCP     | 邮件接收         |
