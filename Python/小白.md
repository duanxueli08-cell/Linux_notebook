
## 一、基础运行环境

### 1. **Python 解释器**

- **安装建议**：

	- 勾选 **“Add Python to PATH”**（非常重要！否则命令行无法识别 `python` 命令）

```powershell
winget install python
py --version

# 如果你需要安装特定版本（如 Python 3.10），可以使用：
winget install Python.Python.3.10
```


### 2. **pip（已内置）**

- Python 官方包管理器，用于安装第三方库（如 `requests`, `numpy`）。
- 自 Python 3.4+ 起默认包含，无需单独安装。

```powershell
pip install requests
pip install -r requirements.txt
```

3. ### **venv（已内置）**
- 用于为不同项目创建独立的 Python 环境，避免包冲突。

```powershell
# 创建虚拟环境（在项目目录下） 
python -m venv myenv 
# 激活（Windows） 
myenv\Scripts\activate 
# 退出 
deactivate
```

## 三、常用辅助工具

### 5. **Git（版本控制）**

- 用于代码管理、协作、回滚。
- 下载：https://git-scm.com/download/win
- 安装后可在 VS Code 或命令行使用 `git` 命令。

### 6. **Windows Terminal（可选但推荐）**

- 微软新一代终端，支持多标签、美化、PowerShell/CMD/WSL。
- 从 Microsoft Store 免费安装。
