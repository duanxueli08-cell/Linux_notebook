
## Copilot
### 1. 前置条件
- 已安装 Obsidian 应用程序（支持 Windows、macOS 和 Linux）。
    
- 注册并获取大模型 API Key和模型名称（目前可以尝试硅基流动或Openrouter，能用到GPT-4o和DeepSeek、Llama、Grok、Claude各大模型）。



### 2.**获取步骤：**
    
    1.访问 [Google AI Studio](https://aistudio.google.com/)。
       访问 [OpenRouter.ai](https://openrouter.ai/)。
        
    2. 登录后进入 **Keys** 页面。
        
    3. 点击 **"Create Key"**，给它起个名字（比如 "Obsidian"）。
        
    4.复制生成的以 `AIza...` 开头的字符串。
       复制生成的以 `sk-or-...` 开头的字符串。
        
- 
  **填写位置：** 在 Copilot 设置 -> `Set Keys` -> 找到 **Google Gemini API Key** 框粘贴。
  **填写位置：** 在 Copilot 设置 -> `Set Keys` -> 找到 **OpenRouter API Key** 框粘贴。

秘钥存放位置：Bitwarden


### 3. 配置 Obsidian Copilot

- 设置 OpenAI API Key

1. 打开 Obsidian，进入 **Copilot Settings（插件设置）**。
    
2. 在 **Basic** 栏中，将 API Key 粘贴到对应字段，并点击 **Verify**

- 选择预置模型和添加模型


总结：
- 拿模型密钥；
- 填入密钥；
- 就这么简单；
